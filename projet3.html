Projet3 : Système de Contrôle d’Accès avec RFID

1. Introduction

Ce projet consiste à réaliser un système de contrôle d’accès intelligent utilisant un badge RFID. Lorsqu’un badge autorisé est présenté devant le capteur, la porte s’ouvre automatiquement grâce à un servo moteur.

Le système affiche les informations sur un écran LCD, utilise des LEDs pour signaler l’état d’accès et un buzzer pour les alertes sonores.

2. Objectifs du projet

  • Lire un badge RFID
  • Vérifier l’identité de l’utilisateur
  • Autoriser ou refuser l’accès
  • Ouvrir automatiquement une porte
  • Afficher les messages sur LCD
  • Utiliser un signal sonore et lumineux

3. Matériels utilisés

  • Carte Arduino Uno
  • Module RFID MFRC522
  • Badge RFID / carte RFID
  • Écran LCD 16x2
  • Servo moteur
  • LED verte
  • LED rouge
  • Buzzer
  • Fils de connexion
  • Breadboard

4. Application utilisée

  • Arduino IDE pour programmer et téléverser le code
  • Moniteur série pour lire les UID des badges
  • Bibliothèques : SPI, MFRC522, LiquidCrystal, Servo

5. Branchement des composants

Module RFID MFRC522

  • SDA / SS → Pin 10
  • RST → Pin 9
  • MOSI → Pin 11
  • MISO → Pin 12
  • SCK → Pin 13
  • VCC → 3.3V
  • GND → GND

LCD 16x2

  • RS → A0
  • E → A1
  • D4 → A2
  • D5 → A3
  • D6 → Pin 2
  • D7 → Pin 3

Servo moteur

  • Signal → Pin 5
  • VCC → 5V
  • GND → GND

LEDs et buzzer

  • LED verte → Pin 6
  • LED rouge → Pin 7
  • Buzzer → Pin 8

6. Principe de fonctionnement

Lorsqu’un badge est détecté, le système lit son UID.

  • Si le badge correspond à l’UID autorisé → accès autorisé
  • La LED verte s’allume
  • Le buzzer émet un bip
  • Le servo moteur ouvre la porte à 90°
  • Après 3 secondes la porte se referme

Si le badge n’est pas reconnu :

  • Message “Accès refusé”
  • LED rouge allumée
  • 3 bips sonores

7. Conclusion

Ce projet montre l’utilisation des systèmes embarqués dans la sécurité et l’automatisation. Il combine électronique, programmation et contrôle d’accès intelligent.
