Projet 2 : Poubelle Intelligente avec ESP32

1. Introduction

Ce projet consiste à réaliser une poubelle intelligente capable de s’ouvrir automatiquement lorsqu’une présence est détectée. Le système utilise une carte ESP32, un capteur ultrason, un écran LCD I2C et un capteur de température et d’humidité. Les informations sont affichées sur un écran LCD et sur une page web grâce à la connexion Wi-Fi.

2. Objectifs du projet

  • Détecter la présence avec un capteur ultrason
  • Ouvrir automatiquement la poubelle si la distance est ≤ 5 cm
  • Afficher "Porte ouverte" ou "Porte fermée" sur l’écran LCD
  • Utiliser une LED verte et une LED rouge pour indiquer l’état
  • Activer un buzzer si la porte reste fermée
  • Afficher les informations sur une page web

3. Matériels utilisés

  • ESP32
  • Capteur Ultrason
  • DHT11 (Température et humidité)
  • Écran LCD I2C
  • LED rouge
  • LED verte
  • Buzzer
  • Résistances
  • Breadboard
  • Fils de connexion
  • Connexion Wi-Fi

4. Applications utilisées

  • Visual Studio Code (VS Code) : éditeur de code utilisé pour écrire et gérer le programme du projet.
  • Extension PlatformIO : extension installée dans VS Code permettant de programmer et téléverser le code vers l’ESP32.
  • Navigateur Web (Google Chrome, Firefox, etc.) : utilisé pour accéder à la page web générée par l’ESP32 et visualiser les données (température, humidité, distance et état de la porte).

5. Principe de fonctionnement

Le capteur ultrason mesure la distance devant la poubelle. Si la distance détectée est inférieure ou égale à 5 cm :

  • La porte s’ouvre
  • La LED verte s’allume
  • L’écran LCD affiche "Porte ouverte"

Si la distance est supérieure à 5 cm :

  • La porte reste fermée
  • La LED rouge s’allume
  • Le buzzer émet un signal sonore
  • L’écran LCD affiche "Porte fermée"

6. Branchement des composants

Capteur Ultrason

  • VCC → 5V
  • GND → GND
  • TRIG → GPIO 5
  • ECHO → GPIO 18

DHT11

  • VCC → 5V
  • GND → GND
  • DATA → GPIO 4

LCD I2C

  • VCC → 5V
  • GND → GND
  • SDA → GPIO 21
  • SCL → GPIO 22

LED rouge

  • Anode → GPIO 19
  • Cathode → GND

LED verte

  • Anode → GPIO 23
  • Cathode → GND

Buzzer

  • + → GPIO 15
  • - → GND

7. Affichage sur la page Web

  • Température
  • Humidité
  • Distance détectée
  • Porte ouverte ou fermée
  • LED rouge ou LED verte

8. Conclusion

Ce projet montre comment utiliser l’ESP32 et plusieurs capteurs pour créer un système automatisé et connecté à Internet. La poubelle intelligente permet d’améliorer l’hygiène et démontre les possibilités de l’Internet des objets.
